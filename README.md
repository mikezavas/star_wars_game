## Обзор игры

**Star Wars: Galactic Missions** - текстовая приключенческая игра во вселенной Star Wars. Игроки отправляются в эпические путешествия, участвуя в космических битвах, проходя обучение у джедаев и выполняя опасные контрабандные рейсы по всей галактике.

## Возможности

### Три типа миссий:
- **Космическая битва** - Сражение со Звездой Смерти
- **Обучение джедая** - Испытания на Дагобе
- **Контрабандный рейс** - Доставка груза на Татуин

### Система прогресса:
- Регистрация и авторизация игроков
- Коллекционирование артефактов (8 уникальных предметов)
- Система рангов: Новичок → Падаван → Рыцарь-джедай → Магистр-джедай
- Сохранение прогресса в JSON файлах

### Боевая система:
- Уникальные корабли с разными характеристиками
- Особые способности для каждого класса кораблей
- Тактические решения во время битв

## Установка

### Требования:
- Python 3.6 или выше
- Стандартные библиотеки Python (без внешних зависимостей)

### Клонирование и запуск:
```bash

git clone https://github.com/ваш-username/star-wars-game.git

# Перейдите в директорию проекта
cd star-wars-game
```

1. Скачайте архив с проектом
2. Распакуйте в удобную директорию
3. Запустите `main.py` через Python

## Классы кораблей

### 1. **StarWarsShip** (базовый класс)
```python
# Основные характеристики
self.title = title      # Название корабля
self.hp = hp            # Здоровье
self.speed = speed      # Скорость
self.damage = damage    # Урон
self.shield = shield    # Щит
self.is_alive = True    # Статус

# Основные методы
attack(target)          # Атаковать цель
get_damage(damage)      # Получить урон
get_heal(heal)          # Восстановить здоровье
```

### 2. **XWing** - Истребитель повстанцев
```python
# Особенности:
# - Торпеды (двойной урон)
# - Ограниченный боезапас: 3 торпеды

# Характеристики:
# HP: 150, Урон: 40, Щит: 50, Торпеды: 3

```

### 3. **TIEFighter** - Истребитель Империи

### 4. **DeathStar** - Звезда Смерти

```python
# Особенности:

```

## Игровой процесс

### Регистрация и вход:
```
STAR WARS: Galactic Missions

1. Регистрация
2. Вход
3. Выход
```

### Главное меню:
```
Игрок: Skywalker
Ранг: Падаван
Миссий: 3
Артефакты: 3/8

Выберите миссию:
1. Космическая битва (сложная)
2. Испытания джедая (средняя)
3. Контрабандный рейс (средняя)
4. Выход
```

### Механики миссий:

#### 1. **Космическая битва**:
- Управление 3 истребителями X-Wing
- Сражение против Звезды Смерти и TIE-истребителей
- Тактический выбор: атаковать истребители или главную цель
- Система раундов и регенерации босса

#### 2. **Обучение джедая**:
- **Тест 1**: Контроль Силы (угадать число от 1 до 7)
- **Тест 2**: Бой на световых мечах против дроида
- Система действий: атака, защита, уклонение
- Логика "камень-ножницы-бумага"

#### 3. **Контрабандный рейс**:
- Управление ресурсами: кредиты, HP корабля, щит
- Покупка улучшений перед вылетом
- Навигация через астероидное поле (3 сектора)
- Выбор стратегии уклонения от патруля

### Артефакты:
1. **Кристалл Кайбера** - Источник энергии для световых мечей
2. **Световой меч** - Оружие джедая
3. **Голокрон джедаев** - Хранилище знаний
4. **Чертежи Звезды Смерти** - Секретные планы
5. **Ключ от храма** - Доступ к древним знаниям
6. **Щит мандалорца** - Легендарная броня
7. **Дроид R2** - Верный помощник
8. **Карта Галактики** - Навигационные данные

## Система рангов

Ранг зависит от количества собранных артефактов:

| Ранг | Артефактов | Описание |
|------|------------|----------|
| **Новичок** | 0-1 | Начинающий искатель приключений |
| **Падаван** | 2-3 | Ученик, изучающий пути Силы |
| **Рыцарь-джедай** | 4-5 | Опытный воин, защитник галактики |
| **Магистр-джедай** | 6+ | Мудрый мастер, собравший все артефакты |
